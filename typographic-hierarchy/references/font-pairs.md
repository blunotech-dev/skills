# Curated Font Pairs for Typographic Hierarchy

Each pair is selected for maximum hierarchy contrast. Import snippets are Google Fonts CDN links.

---

## Editorial / Magazine

### Playfair Display + Source Sans 3
- Display/headings: Playfair Display (700–900, serif)
- Body: Source Sans 3 (300–400, sans-serif)
- Character: Authoritative, editorial, classic
```html
<link href="https://fonts.googleapis.com/css2?family=Playfair+Display:wght@700;900&family=Source+Sans+3:wght@300;400;600&display=swap" rel="stylesheet">
```

### Fraunces + Manrope
- Display/headings: Fraunces (variable, optical sizes)
- Body: Manrope (300–600, geometric)
- Character: Warm, literary, contemporary
```html
<link href="https://fonts.googleapis.com/css2?family=Fraunces:opsz,wght@9..144,300;9..144,700;9..144,900&family=Manrope:wght@300;400;600&display=swap" rel="stylesheet">
```
